import { Database, Book, PlusCircle, Bot, Filter, Eraser, Save, Share } from "lucide-react";

# Sidebar

The sidebar is your gateway to unlocking the full potential of grafiki! It's divided into two main sections: **Dataset and Template Database** and **Actions**.

## Database

Here you can access the dataset and template database respectively.

| Icon                                     | Action                                                                    |
| ---------------------------------------- | ------------------------------------------------------------------------- |
| <Database style={{display: 'inline'}} /> | Opens the dataset section where you can view and manage your datasets.    |
| <Book style={{display: 'inline'}} />     | Opens the template database where you can view and manage your templates. |

## Actions

These buttons are used to perform various actions in the dashboard.

| Icon                                       | Action                                                             |
| ------------------------------------------ | ------------------------------------------------------------------ |
| <PlusCircle style={{display: 'inline'}} /> | Adds new items to the dashboard.                                   |
| <Bot style={{display: 'inline'}} />        | Opens the assistant section where you can get help and assistance. |
| <Filter style={{display: 'inline'}} />     | Filters the items in the dashboard based on specific criteria.     |
| <Eraser style={{display: 'inline'}} />     | Clears all the items in the dashboard.                             |
| <Save style={{display: 'inline'}} />       | Saves the current state of the dashboard.                          |
| <Share style={{display: 'inline'}} />      | Shares the dashboard with others.                                  |
